# Customs Compliance Expertise

This skill provides comprehensive knowledge of customs procedures, tariff classification, duty calculation, trade agreements, and regulatory compliance for international shipping.

## HS Code Classification

### Harmonized System Structure

The Harmonized System (HS) is an international nomenclature for classifying traded products.

```
8471.30.0100
│││  │  └── US statistical suffix (4 digits)
│││  └───── HS subheading (2 digits) - international
││└──────── HS heading (2 digits) - international
└└───────── HS chapter (2 digits) - international
```

**Classification Levels:**
| Level | Digits | Scope |
|-------|--------|-------|
| Chapter | 2 | Broad category (e.g., 84 = Machinery) |
| Heading | 4 | Product group (e.g., 8471 = Computers) |
| Subheading | 6 | International standard |
| National | 8-10 | Country-specific detail |

### General Rules of Interpretation (GRI)

| Rule | Application |
|------|-------------|
| GRI 1 | Classify by section/chapter notes and heading terms |
| GRI 2(a) | Incomplete/unfinished goods classify as complete if essential character |
| GRI 2(b) | Mixtures/combinations - classify by material giving essential character |
| GRI 3(a) | Most specific heading prevails |
| GRI 3(b) | Composite goods - material giving essential character |
| GRI 3(c) | Last in numerical order |
| GRI 4 | Most akin to similar goods |
| GRI 5 | Cases/containers classified with contents |
| GRI 6 | Apply above rules at subheading level |

### Common HS Chapters

| Chapter | Description | Example Products |
|---------|-------------|------------------|
| 39 | Plastics | Plastic containers, films, housewares |
| 61 | Knitted apparel | T-shirts, sweaters, underwear |
| 62 | Woven apparel | Suits, shirts, dresses |
| 64 | Footwear | Shoes, boots, sandals |
| 84 | Machinery | Engines, pumps, computers |
| 85 | Electrical | Motors, phones, TVs |
| 87 | Vehicles | Cars, trucks, parts |
| 94 | Furniture | Chairs, tables, mattresses |
| 95 | Toys/Games | Toys, sports equipment |

### Classification Best Practices

1. **Read the product description** - Understand composition, function, use
2. **Check section and chapter notes** - Legal notes override headings
3. **Identify essential character** - What makes this product what it is?
4. **Apply GRI systematically** - Don't skip to GRI 3 without checking GRI 1-2
5. **Check binding rulings** - Prior CBP determinations for similar products
6. **Document reasoning** - Create classification worksheet

### Import Timeline

| Event | Deadline | Notes |
|-------|----------|-------|
| ISF Filing | 24 hrs before vessel departure | 10+2 data elements |
| Entry Filing | 15 days after arrival | Or 15 days after FTZ admission |
| Entry Summary | 10 working days after entry | With duty payment |
| Liquidation | ~314 days after entry | Final duty determination |
| Protest | 180 days after liquidation | Challenge duty assessment |
| Drawback | 5 years | Claim refund on re-export |

### ISF (10+2) Requirements

**Importer Security Filing (10 data elements):**
1. Manufacturer name/address
2. Seller name/address
3. Buyer name/address
4. Ship-to name/address
5. Container stuffing location
6. Consolidator name/address
7. Importer of record number
8. Consignee number
9. Country of origin
10. HS tariff number (6-digit)

**Carrier Requirements (2 elements):**
1. Vessel stow plan
2. Container status messages

**Penalties:** $5,000 per violation for late/inaccurate ISF

### Customs Bond

**Types:**
| Bond Type | Coverage | Use |
|-----------|----------|-----|
| Single Entry | One shipment | Occasional importers |
| Continuous | One year, all entries | Regular importers |

**Bond Amount:**
- Minimum: $50,000
- Formula: 10% of duties + taxes + fees paid prior year
- Minimum $50,000 or calculated amount, whichever greater

## Duty Calculation

### Customs Valuation

**Transaction Value** (primary method):
```
Customs Value = Price Paid + Additions - Deductions
```

**Additions:**
- Packing costs
- Selling commissions
- Assists (tooling, materials provided)
- Royalties/license fees
- Proceeds of resale accruing to seller

**Deductions:**
- Freight after importation
- Insurance after importation
- Duties and taxes
- Installation/assembly costs

### Duty Types

| Type | Calculation | Example |
|------|-------------|---------|
| Ad Valorem | % of value | 5% of $10,000 = $500 |
| Specific | Per unit | $0.50/kg × 1000kg = $500 |
| Compound | Both | 5% + $0.10/kg |

### Additional Fees

| Fee | Rate | Notes |
|-----|------|-------|
| Merchandise Processing Fee (MPF) | 0.3464% | Min $29.66, Max $575.35 |
| Harbor Maintenance Fee (HMF) | 0.125% | Ocean shipments only |
| Section 301 (China) | 7.5-25% | Additional on List 1-4 products |
| Section 232 (Steel/Aluminum) | 25%/10% | Unless exempted |
| Antidumping Duty (AD) | Varies | Product/country specific |
| Countervailing Duty (CVD) | Varies | Product/country specific |

### Duty Calculation Example

```
Product: Wireless headphones from China
HS Code: 8518.30.2000
Declared Value: $50,000

Base Duty Rate: 0%
Section 301 (List 3): 25%

Calculation:
Base Duty: $50,000 × 0% = $0
Section 301: $50,000 × 25% = $12,500
MPF: $50,000 × 0.3464% = $173.20
HMF: $50,000 × 0.125% = $62.50

Total: $12,735.70
Effective Rate: 25.47%
```

## Trade Agreements

### USMCA (US-Mexico-Canada)

**Benefits:** Duty-free treatment for qualifying goods

**Requirements:**
- Certificate of Origin (can be self-certified)
- Meet Rules of Origin criteria
- Direct shipment (no transshipment through non-parties)

**Rules of Origin Types:**
| Rule | Requirement |
|------|-------------|
| Wholly Obtained | Grown/extracted in region |
| Tariff Shift | Non-originating materials change HS classification |
| Regional Value Content | Minimum % of value from region |

**RVC Calculation Methods:**
```
Transaction Value Method:
RVC = [(TV - VNM) / TV] × 100

Net Cost Method:
RVC = [(NC - VNM) / NC] × 100

Where:
TV = Transaction Value
NC = Net Cost
VNM = Value of Non-originating Materials
```

### Generalized System of Preferences (GSP)

- Duty-free for eligible products from developing countries
- Product must be on GSP-eligible list
- Must meet 35% value-added requirement
- Direct shipment required

## Compliance Best Practices

### Record Keeping

**Retention Period:** 5 years from date of entry

**Required Records:**
- Entry summary and supporting documents
- Commercial invoices
- Packing lists
- Bills of lading
- Certificates of origin
- Payment records
- Correspondence

### Internal Controls

| Control | Purpose |
|---------|---------|
| Classification database | Consistent HS codes |
| Valuation procedures | Accurate duty calculation |
| Restricted party screening | Sanctions compliance |
| FTA qualification | Proper origin claims |
| Record retention | Audit readiness |
| Training | Staff competency |

### Common Compliance Errors

| Error | Risk | Prevention |
|-------|------|------------|
| Misclassification | Wrong duty rate | Classification review process |
| Undervaluation | Fraud penalties | Proper assists/additions |
| Origin errors | FTA penalties | Verify supplier certifications |
| Late ISF | $5,000 penalty | Automated filing triggers |
| Missing licenses | Seizure, fines | Export classification review |

## Penalties

### Civil Penalties

| Violation | Penalty |
|-----------|---------|
| Negligence | 2× revenue loss or 20% of value |
| Gross negligence | 4× revenue loss or 40% of value |
| Fraud | Domestic value of goods |

### Mitigation Factors

- Prior disclosure (reduces penalties significantly)
- Compliance program in place
- Immediate corrective action
- Cooperation with CBP
- No prior violations
